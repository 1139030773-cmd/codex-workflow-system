metadata:

---

验证某个东西是否有效时，必须**直接执行实操**（跑命令、调接口、装插件），不能只靠搜索文档 + 推测。

**Why:** 用户发现 AI 用搜索文档推测代替实操验证，认为这是严重偷懒。文档可能过时、不完整，实操结果才是真相。用户自己验证过没有问题，AI 却基于文档推测说有问题，浪费时间且失去信任。

**How to apply:** 
- 需要验证 → 先想能不能直接跑一下/调一下/试一下
- 能实操的不搜文档，搜了文档也必须有实操佐证
- 无法实操时明确说"我无法直接验证，需要你帮忙试"而不是假装通过搜索确认了
- 相关：[[session-recovery]] [[active-task]]
